# Prachi Panditrao - Portfolio

**[🌐 View My Live Portfolio Here](https://prachipanditrao.github.io)**

Welcome to my personal portfolio repository! This site showcases my software engineering projects and experience.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Mobile Navigation**: Hamburger menu that closes on link selection or outside tap
- **Theme Support**: Automatically adapts to browser's light/dark theme preference using CSS `prefers-color-scheme`
- **Smooth Scrolling**: Enhanced navigation experience
- **Contact Integration**: Direct links to email, LinkedIn, and other platforms
- **Project Showcase**: Grid layout for featured work

## Technologies Used

- **HTML5**: Semantic markup for structure
- **CSS3**: Custom properties for theming, flexbox/grid layouts, responsive design
- **JavaScript (Vanilla)**: Mobile menu interactions and dynamic elements

## Development Tools

- **GitHub MCP**: Used for version control and repository management
- **Figma MCP**: Integrated for design-to-code workflow and asset management

Open `index.html` in your web browser to view the portfolio.

## Files

- `index.html` — Main landing page with hero section, skills, expertise, projects, and contact
- `styles.css` — Responsive styling with light/dark theme support
- `script.js` — Mobile navigation toggle, menu close handlers, and footer year injection

## Usage

1. Clone or download the repository
2. Open `index.html` in any modern web browser
3. The site will automatically detect your browser's theme preference

No build process or dependencies required — it's a pure static site.
